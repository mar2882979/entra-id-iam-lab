# Microsoft Entra ID IAM Lab

## Overview

This repository documents my hands-on learning journey in Identity and Access Management (IAM) using Microsoft Entra ID.

The goal of these labs is to develop practical cybersecurity skills through real-world IAM tasks and administrative exercises.

## Technologies Used

* Microsoft Entra ID
* Microsoft Azure Portal
* Microsoft 365 Developer Environment

## Skills Being Developed

* Identity and Access Management (IAM)
* User Provisioning and Deprovisioning
* Role-Based Access Control (RBAC)
* Multi-Factor Authentication (MFA)
* Conditional Access
* Identity Governance
* Cloud Security Fundamentals
